## .ACCDR вариант №

Файл с расширением .accdr представляет собой файл базы данных времени выполнения, созданный с помощью базы данных Microsoft Access, которая используется для создания реляционных баз данных. Он создается как файл базы данных только для чтения, который можно открыть только для просмотра и нельзя редактировать. Microsoft Access также позволяет создавать другие типы файлов базы данных, такие как [ACCDB](/ru/database/accdb/), [ACCDC](/ru/database/accdc/) и [ACCDT](/ru/database/accdt/). Файлы ACCDR на самом деле являются файлами ACCDB, но без возможности добавления, обновления или удаления данных из этих файлов. Файлы ACCDR можно открывать с помощью Microsoft Access и Office 365.

## Формат файла ACCDR — дополнительная информация

Файлы ACCDR — это двоичные файлы, имеющие ту же структуру, что и файлы ACCDB, но без возможности обновления содержимого базы данных. Из исследований ясно, что Microsoft применяет некоторые специальные характеристики к папке Program Files, которые не позволяют вносить изменения в файлы ACCDR. Если те же самые файлы ACCDR переименовываются в ACCDB, эти файлы становятся редактируемыми.
