## .FP7 вариант №

Файл с расширением .fp7 — это файл базы данных, созданный с помощью FileMaker Pro версий 7–11. Он хранит данные в таблицах с записями, где каждая запись содержит одно или несколько полей. Файлы FP7 могут хранить большие данные, включая данные личных баз данных и организационные записи. Более новой версией FileMaker Pro является [FMP12](/ru/database/fmp12/), которая была представлена с запуском FileMake Pro версии 12. Помимо текстовых данных, файл FP7 может также содержать формы, диаграммы и другие объекты базы данных. . Файлы FP7 можно открывать с помощью FileMaker Pro Advanced.

## Формат файла FP7

Файлы FP7 сохраняются на диск как двоичные файлы в собственном формате файлов FileMaker. Подробная информация о внутреннем формате файла формата файла FP7 публично недоступна.
