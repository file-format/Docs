## .GPX вариант №

Файлы с расширением GPX представляют собой формат обмена данными GPS для обмена данными GPS между приложениями и веб-службами в Интернете. Это облегченный формат XML-файла, который содержит данные GPS, т. е. путевые точки, маршруты и треки, которые необходимо импортировать и отображать несколькими программами. Формат файла GPX является открытым и поддерживается различными приложениями и устройствами GPS. Данные GPS из таких файлов можно загружать для отображения в картографических приложениях для геопространственных целей.

## Формат файла GPX ##

Файл GPX состоит из данных о местоположении широты и долготы, значений высоты и другой, возможно, другой описательной информации. Данные о местоположении выражаются в десятичных градусах, а высота — в метрах. Время в файле GPX указано в формате универсального координированного времени (UTC) с использованием формата ISO 8601. Преимущества использования файла GPX заключаются в следующем:

* GPX позволяет обмениваться данными с постоянно растущим списком программ для Windows, MacOS, Linux, Palm и PocketPC.
* GPX можно преобразовать в файлы других форматов с помощью простой веб-страницы или программы-конвертера.
* GPX основан на стандарте XML, поэтому многие из новых программ, которые вы используете (например, Microsoft Excel), могут читать файлы GPX.
* GPX позволяет любому человеку в Интернете легко разрабатывать новые функции, которые будут мгновенно работать с вашими любимыми программами.

[Схема GPX](https://www.topografix.com/GPX/1/1/gpx.xsd) показывает представление формата файла GPX для справки.

### Основные данные ###

Ниже приведены основные данные, которые являются частью файла GPX для представления данных GPS.

* **Путевые точки**: Путевая точка представляет собой координаты WGS84 (GPS) точки и представляет слой функций типа OGR wkbPoint.
* **Маршруты**: представляют слой объектов типа OGR wkbLineString. Он включает в себя список точек трека, которые представляют собой путевые точки, показывающие поворот или точки этапа, которые ведут к месту назначения.
* **Треки**: Треки представляют собой слой признаков OGR типа wkbMultiLineString. Он состоит как минимум из одного сегмента, содержащего путевые точки в упорядоченном списке точек, описывающих путь. Он состоит из списка точек трека, которые представляют собой непрерывный трек GPS.

### Пример файла GPX ###

Следующий файл GPX показывает организацию данных GPS в файле GPX и может дать хорошее представление о содержимом файла GPX.

```
<gpx xmlns#"http://www.topografix.com/GPX/1/1" xmlns:gpxx#"http://www.garmin.com/xmlschemas/GpxExtensions/v3" xmlns:gpxtpx#"http://www.garmin.com/xmlschemas/TrackPointExtension/v1" creator#"Oregon 400t" version#"1.1" xmlns:xsi#"http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation#"http://www.topografix.com/GPX/1/1 http://www.topografix.com/GPX/1/1/gpx.xsd http://www.garmin.com/xmlschemas/GpxExtensions/v3 http://www.garmin.com/xmlschemas/GpxExtensionsv3.xsd http://www.garmin.com/xmlschemas/TrackPointExtension/v1 http://www.garmin.com/xmlschemas/TrackPointExtensionv1.xsd">
  <metadata>
    <link href#"http://www.garmin.com">
      <text>Garmin International</text>
    </link>
    <time>2009-10-17T22:58:43Z</time>
  </metadata>
  <trk>
    <name>Example GPX Document</name>
    <trkseg>
      <trkpt lat#"47.644548" lon#"-122.326897">
        <ele>4.46</ele>
        <time>2009-10-17T18:37:26Z</time>
      </trkpt>
      <trkpt lat#"47.644548" lon#"-122.326897">
        <ele>4.94</ele>
        <time>2009-10-17T18:37:31Z</time>
      </trkpt>
      <trkpt lat#"47.644548" lon#"-122.326897">
        <ele>6.87</ele>
        <time>2009-10-17T18:37:34Z</time>
      </trkpt>
    </trkseg>
  </trk>
</gpx>
```
