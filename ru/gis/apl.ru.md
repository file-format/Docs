## .APL вариант №

Файл с расширением .apl представляет собой файл слоя карты, используемый прикладным программным обеспечением ESRI ArcPad. Он содержит информацию о подмножестве свойств слоя карты для экспортированной карты ArcGIS. Он используется полевым персоналом для захвата, анализа и отображения географической информации в виде растровых и векторных изображений. Он включает в себя символы для отображения объектов, масштабов и выражений надписей для слоя карты. Файлы APL можно открывать с помощью ArcPad Pro на портативных устройствах, используемых в полевых работах.

## Формат файла APL

APL — это бинарные файлы ГИС, информация о формате которых недоступна для общественности.
