## .3DC вариант №

Файл 3DC — это файл данных GPS, используемый программным обеспечением GPS-навигации [iGO] (https://en.wikipedia.org/wiki/IGO_(software)). Он содержит информацию о трехмерных данных здания, а также метаданные для отображения в приложении. Файлы 3DC сохраняются на карту памяти навигационного блока и в основном хранятся в каталоге \​iGO\​content\​building\​.

## О навигации iGO

Программное обеспечение iGO используется для целей навигации. Он получает информацию о местоположении пользователя с помощью GPS-устройств и помогает в навигации с помощью 3D-карт и маршрутов. Он доступен для Android, iOS и других популярных портативных устройств. Помимо навигационных функций, программное обеспечение iGO также обеспечивает автономное хранение карт на случай, если пользователям будет сложно подключиться к Интернету для получения картографических данных в реальном времени.
