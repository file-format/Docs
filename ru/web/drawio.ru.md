## .DRAWIO вариант №

Файл с расширением .drawio представляет собой файл чертежа, созданный с помощью [diagrams.net](https://www.diagrams.net/) draw.io, которая представляет собой программу с открытым исходным кодом для работы с диаграммами. Он содержит общую информацию о содержимом и форматировании элементов диаграммы, таких как текст, изображения, макет, формы и расположение. Диаграммы, поддерживаемые DRAWIO, включают блок-схемы, организационные диаграммы, карты, инженерные элементы, диаграммы процессов, диаграммы и многое другое. Файлы DRAWIO можно экспортировать в различные форматы, такие как [JPG](/ru/image/jpeg/), [PNG](/ru/image/png/), [BMP](/ru/image/bmp/), [XML](/ru/web/xml/), PDF, [HTML](/ru/web/html/) и [VSDX](/ru/visio/vsdx/).

## Формат файла DRAWIO

Файлы DRAWIO представляют собой файлы векторных изображений, хранящиеся в стандартном формате файлов XML. Разработанный диаграммами.net, он предоставляет возможность хранить информацию о диаграммах, аналогичную Microsoft Visio. DrawIO доступен как [онлайн-приложение](https://app.diagrams.net/) для создания, открытия и экспорта диаграмм в различные форматы. Приложение основано на библиотеке диаграмм mxGraph, которая предоставляет приложения для создания интерактивных графиков и диаграмм, которые работают в любом основном браузере, таком как Chrome, Firefox, Edge и Safari.
