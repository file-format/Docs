## .MB вариант №

Файл с расширением .mb представляет собой двоичный файл проекта, созданный с помощью приложения Autodesk Maya. В отличие от формата файла [MA](/ru/3d/ma/), который находится в формате файла ASCII, файлы MB хранятся в двоичном формате. Их нельзя открыть в текстовых редакторах, но можно просмотреть в шестнадцатеричных редакторах. Файлы MB содержат такую информацию, как геометрия, освещение, анимация и визуализация для определения 3D-сцен.

Autodesk Maya 2020 можно использовать для открытия файлов .mb.

## Формат файла МБ

Файлы MB имеют двоичный формат файла, который является сжатым форматом файла и может быть открыт только поддерживающим приложением. Внутренние сведения о формате файла MB недоступны и могут быть обработаны только каким-либо сценарием для их распаковки.
