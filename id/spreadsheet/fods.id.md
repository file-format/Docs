## Apa itu file FODS?

File dengan ekstensi .fods adalah jenis format dokumen OpenDocument Spreadsheet yang menyimpan data dalam baris dan kolom. Format ditentukan sebagai bagian dari spesifikasi ODF 1.2 yang diterbitkan dan dikelola oleh OASIS. File FODS tidak dapat dibuka dengan Excel, aplikasi perangkat lunak Spreadsheet lain dari Microsoft. File FODS dapat disimpan sebagai [ODS](/id/spreadsheet/ods/) menggunakan LibreOffice dan dapat dikonversi ke format lain seperti [XLS](/id/spreadsheet/xls/) dan [XLSX](/id/spreadsheet/xlsx/).

## Format File FODS

OpenDocument menentukan standar [ZIP](/id/compression/zip/) serta [XML](/id/web/xml/) untuk format file yang didukungnya. Format file FODS didasarkan pada format dokumen XML tunggal, juga dikenal sebagai Flat XML atau file XML yang tidak terkompresi. Penggunaan format zip lebih umum daripada format OpenDocument berbasis XML seperti di [ODS](/id/spreadsheet/ods/). Spesifikasi teknis OpenDocument tidak mencantumkan satu pun dokumen XML dan rincian lebih lanjut dari dokumen tersebut tidak tersedia.
