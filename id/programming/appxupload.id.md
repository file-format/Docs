## Apa itu file APPXUPLOAD?

File APPXUPLOAD adalah file paket tunggal yang berisi beberapa paket aplikasi atau bundel aplikasi untuk diunggah ke Microsoft Store. Ini dihasilkan secara otomatis oleh [Microsoft Visual Studio](https://visualstudio.microsoft.com/) saat pengembang mengemas aplikasi mereka untuk dipublikasikan ke [Microsoft Store](https://www.microsoft.com/en-us/ toko/aplikasi/windows). Selain file [APPX](/id/programming/appx/), file APPXUPLOAD berisi file simbol yang menganalisis kinerja aplikasi setelah dipublikasikan ke Microsoft Store.
