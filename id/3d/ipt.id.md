## Apa itu file IPT?

File dengan ekstensi .ipt adalah format file Inventor Part asli Autodesk untuk suku cadang.
![IPT File](../ipt2.jpg "IPT File")
Ini digunakan dalam kombinasi dengan file rakitan Autodesk (.iam). File IPT dapat diimpor dalam 3DS Max sebagai Objek Tubuh di mana geometri dalam format padatan ACIS tetap dalam format yang sama. 3DS Max mempertahankan penamaan objek seperti yang ditugaskan di Autodesk Inventor dan model komponen dapat diedit mirip dengan objek lain termasuk menerapkan pengubah, mengubah bahan, menambahkan pencahayaan dan kamera, membuat animasi, dan sebagainya. Anda dapat membuka file IPT dengan aplikasi seperti Autodesk Inventor dan Autodesk Fusion.

## Format File IPT

Autodesk **ekstensi file IPT** digunakan untuk menyimpan file yang terdiri dari sketsa, fitur, dan badan yang membuat bagian tersebut dapat digunakan dalam rakitan. Sketsa adalah komponen yang paling banyak digunakan dan sebagian besar dimulai dengan sketsa. Ini adalah profil fitur dan geometri apa pun yang diperlukan untuk membuat fitur tersebut. Berbagai fitur membentuk `model bagian` dan bagian multi-tubuh dapat berbagi fitur. Hubungan geometris seperti paralel dan tegak lurus dikendalikan oleh kendala sketsa. Pengaruh modifikasi dapat diamati dengan menyesuaikan kendala atau parameter dimensi untuk mengontrol ukuran dan bentuk model.
![IPT File Format](../ipt.jpg "IPT File Format")

**Catatan:** Spesifikasi format file IPT tidak tersedia untuk umum.
