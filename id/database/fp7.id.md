## Apa itu file FP7?

File dengan ekstensi .fp7 adalah file database yang dibuat dengan FileMaker Pro versi 7-11. Ini menyimpan data dalam tabel dengan catatan di mana setiap catatan berisi satu atau lebih bidang. File FP7 memiliki kemampuan untuk menyimpan data besar termasuk data database pribadi dan catatan organisasi. Versi FileMaker Pro yang lebih baru adalah [FMP12](/id/database/fmp12/) yang diperkenalkan dengan peluncuran FileMake Pro versi 12. Selain data tekstual, file FP7 juga dapat menyimpan formulir, bagan, dan objek database lainnya . File FP7 dapat dibuka dengan FileMaker Pro Advanced.

## Format File FP7

File FP7 disimpan ke disk sebagai file biner dalam format file milik FileMaker. Detail format file internal dari format file FP7 tidak tersedia untuk umum.
