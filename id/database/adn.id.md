## Apa itu file ADN?

File dengan ekstensi .adn adalah file template kosong Microsoft Access yang digunakan untuk membuat file proyek Access dengan pengaturan yang telah disimpan sebelumnya. File proyek Microsoft Access disimpan sebagai file ADP dan berisi informasi seperti Koneksi Database dan pengaturan startup proyek. File ADN tersedia untuk versi Microsoft Access 2007 dan 2010. Microsoft Access mendukung format file lain seperti [MDB](/id/database/mdb/), [ACCDB](/id/database/accdb/), [ACCDC](/id/database/accdc/), [ACCDE](/id/database/accde/ ), dan masih banyak lagi selain file ADN ini. File ADN dapat dibuka menggunakan Office 365 dan Microsoft Access.

## Format File ADN - Informasi Lebih Lanjut

ADN adalah file template database yang disimpan ke disk sebagai file biner. Detail format file internal file ADN tidak tersedia untuk referensi pengembang.
