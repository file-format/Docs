## Apa itu file ACCDR?

File dengan ekstensi .accdr adalah file database Runtime yang dibuat dengan database Microsoft Access yang digunakan untuk membuat database relasional. Itu dibuat sebagai file database hanya-baca yang dapat dibuka hanya untuk dilihat dan tidak dapat diedit. Microsoft Access juga memungkinkan untuk membuat jenis file database lainnya seperti [ACCDB](/id/database/accdb/), [ACCDC](/id/database/accdc/), dan [ACCDT](/id/database/accdt/). File ACCDR sebenarnya adalah file ACCDB, tetapi tanpa ketentuan untuk menambah, memperbarui, atau menghapus data dari file-file ini. File ACCDR dapat dibuka dengan Microsoft Access dan Office 365.

## Format File ACCDR - Informasi Lebih Lanjut

File ACCDR adalah file biner yang memiliki struktur yang sama dengan file ACCDB tetapi tanpa ketentuan untuk memperbarui konten database. Dari penelitian, jelas bahwa Microsoft menerapkan beberapa karakteristik khusus pada folder Program Files yang tidak mengizinkan perubahan dilakukan pada file ACCDR. Jika file ACCDR yang sama diubah namanya menjadi ACCDB, file ini menjadi dapat diedit.
