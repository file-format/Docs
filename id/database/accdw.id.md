## Apa itu file ACCDW?

File dengan ekstensi .accdw adalah file tautan yang dibuat dengan Microsoft Access dan berisi informasi tentang tautan untuk mengunduh file database Access yaitu [ACCDB](/id/database/accdb/) dari server SharePoint. Ini memungkinkan pengguna untuk berbagi file database yang dihosting dari jarak jauh. Membuka file ACCDW mengunduh file ACCDB tertaut sebagai salinan lokal di komputer pengguna. File yang diunduh disimpan ke lokasi unduhan default dan dapat diakses dengan menavigasi ke sana. Biasanya, file-file ini diunduh dan disimpan dengan nama "SiteServer.accdb" yang merujuk ke database klien.

## Format File ACCDW - Informasi Lebih Lanjut

File ACCDW adalah file XML yang menyediakan tautan ke situs SharePoint tempat Layanan Access dihosting. Ini hanya jalan pintas dan membutuhkan koneksi internet untuk menjalankannya. Dalam hal online, SharePoint di-cache secara lokal untuk memberikan opsi untuk menjadikannya offline nanti.
