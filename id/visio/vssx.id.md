## Apa itu file VSSX?

File dengan ekstensi .vssx adalah stensil gambar yang dibuat dengan [Microsoft Visio](https://products.office.com/en/visio/flowchart-software) 2013 dan yang lebih baru. Format file VSSX dapat dibuka dengan Visio 2013 ke atas. File Visio dikenal untuk representasi dari berbagai elemen gambar seperti kumpulan bentuk, konektor, diagram alur, tata letak jaringan, diagram UML, diagram perangkat lunak, model database, pemetaan objek, dan informasi serupa lainnya. Microsoft Visio juga menyediakan kemampuan untuk mengonversi file Visio ke format file yang berbeda seperti [PNG](/id/image/png/), [BMP](/id/image/bmp/), [PDF](/id/pdf/) dan lainnya. Ini tersedia untuk Windows dan Mac OS.

## Format File VSSX

Format file VSSX didasarkan pada format OpenOffice yang diadopsi oleh Microsoft sejak 2007. Ini didasarkan pada arsip [ZIP](/id/compression/zip/) yang mewakili format file keseluruhan berdasarkan spesifikasi format file XML. Format file biner yang setara dengan VSSX adalah VSS yang didukung hingga Visio 2007. Anda dapat melihat konten format file VSSX dengan mengganti ekstensinya dengan .ZIP dan membuka dalam format file pengarsipan seperti WinZIP.

File VSDX didasarkan pada Open Packaging Conventions dan XML dan pengembang bisa mendapatkan keuntungan dari format ini dengan mempelajari cara bekerja dengan file ini secara terprogram. Format ini mewarisi banyak struktur XML yang sama sebagai bagiannya dari format file Visio XML Drawing (.vdx). Interoperabilitas dengan file Visio sangat meningkat karena perangkat lunak pihak ketiga dapat memanipulasi file Visio pada tingkat format file.

Jenis file tertentu lainnya yang terdiri dari format file Visio 2013 meliputi:

* .vsdm (gambar berkemampuan makro Visio)
* .vssx (stensil Visio)
* .vssm (stensil berkemampuan makro Visio)
* .vstx (templat Visio)
* .vstm (template berkemampuan makro Visio)

Setiap file Visio disebut sebagai paket yang menampung file atau bagian lain. Bagian paket dapat berupa file XML, gambar, atau bahkan solusi VBA. Bagian dalam paket dapat dibagi menjadi bagian "dokumen" dan "hubungan".
