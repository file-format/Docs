## Apa itu file DISCOMAP?

File DISCOMAP adalah file keluaran yang dibuat oleh Alat Penemuan Layanan Web Microsoft, **Disco.exe**. Ini berisi hasil yang ditemukan Layanan Web ASP.NET di URL tertentu. Contoh umum dari layanan yang ditemukan tersebut mencakup lokasi file .DISCO atau .WSDL. Saat alat Microsoft Web Discovery berjalan, ia mencari informasi penemuan berdasarkan konten XML yang ditemukan di file DISCO. Hasilnya disimpan dalam file DISCOMAP yang dapat dibuka dengan Microsoft Visual Studio IDE.
