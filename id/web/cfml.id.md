## Apa itu file CFML?

File dengan ekstensi .cfml adalah file ColdFusion Markup Language yang digunakan untuk membuat halaman web. Ini mengacu pada seperangkat aturan yang digunakan untuk menentukan bagaimana aplikasi ColdFusion akan dikembangkan oleh programmer. ColdFusion adalah bahasa pemrograman yang digunakan untuk membuat aplikasi web sisi server dengan cepat dan kurang dari teknologi lain seperti [ASP](/id/web/asp/), [PHP](/id/programming/php/), dll. Beberapa aplikasi yang bisa membuka file CML antara lain Adobe ColdFusion 2018, Adobe ColdFusion Builder, dan Adobe Dreamweaver.

## Format File CFML - Informasi Lebih Lanjut

File CFML adalah file markup dan berisi elemen web dalam bentuk tag. Ini adalah file teks yang dapat dibuka dan diedit di editor teks apa pun. File CFML terdiri dari beberapa tag yang mirip dengan [HTML](/id/web/html/) dan biasanya terdiri dari tag pembuka dan penutup. Tag ini juga dapat berisi satu atau lebih atribut.

### Tag Sintaks

Berikut ini adalah sintaks umum dari tag CFML.

```
<tagname>
  Code/text that is affected by the surrounding tags.
</tagname>
```

Sebagian besar tag menerima atribut dan didefinisikan sebagai berikut.

```
<tagname attribute="value">
  Code/text that is affected by the surrounding tags.
</tagname>
```

Beberapa tag ini juga menerima banyak atribut dengan sintaks berikut.

```
<tagname attribute1="value" attribute2="value">
  Code/text that is affected by the surrounding tags.
</tagname>
```

## Contoh Kode CFML

Berikut adalah contoh yang menggunakan tag CFML sebenarnya - tag `cfoutput`:

```
<cfoutput>
  #firstname#
</cfoutput>
```
