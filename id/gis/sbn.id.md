## Apa itu file SBN?

File dengan ekstensi .sbn adalah file indeks spasial yang terkait dengan file ESRI ArcGIS [.shp](/id/gis/shp/). Ini digunakan untuk mengoptimalkan kueri spasial ketika indeks telah dilakukan pada data. Jenis file lain yang digunakan bersama .sbn adalah file .sbx. File SBN dan SBX bersama-sama membentuk indeks bentuk untuk mempercepat kueri spasial. File SBN bersifat opsional dan dapat dibuka dengan [ESRI ArcGIS Pro](https://www.esri.com/en-us/arcgis/products/arcgis-pro/overview).

## Format File SBN - Informasi Lebih Lanjut

File SBN disimpan ke disk sebagai file biner dan detail format file internalnya tidak tersedia. Ini menyimpan representasi biner dari file SHP untuk kueri spasial. File indeks SBX digunakan bersama file SBN untuk mempercepat kueri spasial pada shapefile.
