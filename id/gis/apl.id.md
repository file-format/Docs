## Apa itu file APL?

File dengan ekstensi .apl adalah file lapisan Peta yang digunakan oleh perangkat lunak aplikasi ESRI ArcPad. Ini berisi informasi tentang subset properti lapisan peta untuk peta ArcGIS yang diekspor. Ini digunakan oleh personel berbasis lapangan untuk menangkap, menganalisis, dan menampilkan informasi geografis dalam bentuk gambar raster dan vektor. Ini mencakup simbologi untuk menampilkan fitur, skala, dan ekspresi label untuk lapisan peta. File APL dapat dibuka dengan ArcPad Pro pada perangkat genggam yang digunakan dalam pekerjaan lapangan.

## Format File APL

APL adalah file GIS biner dan informasi format filenya tidak tersedia untuk umum.
