## Mi az a DISCOMAP fájl?

A DISCOMAP-fájl egy kimeneti fájl, amelyet a Microsoft Web Services Discovery Tool (**Disco.exe**) hoz létre. Tartalmazza azokat az eredményeket, amelyek az ASP.NET webszolgáltatások egy adott URL-címen találhatók. Az ilyen felfedezett szolgáltatások gyakori példái közé tartozik a .DISCO vagy .WSDL fájlok helye. Amikor a Microsoft Web Discovery eszköz fut, a DISCO-fájlban található XML-tartalom alapján keres információt. Az eredményeket a DISCOMAP fájl tárolja, amely a Microsoft Visual Studio IDE programmal nyitható meg.
