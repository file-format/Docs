## 什么是一 .fods 文件？

具有 .fods 扩展名的文件是一种 OpenDocument 电子表格文档格式，可将数据存储在行和列中。该格式被指定为 OASIS 发布和维护的 ODF 1.2 规范的一部分。 FODS 文件无法使用 Excel（Microsoft 的另一个电子表格软件应用程序）打开。 FODS 文件可以使用 LibreOffice 保存为 [ODS](/zh/spreadsheet/ods/)，并且可以转换为其他格式，例如 [XLS](/zh/spreadsheet/xls/) 和 [XLSX](/zh/spreadsheet/xlsx/)。

## FODS 文件格式

OpenDocument 为其支持的文件格式定义了 [ZIP](/zh/compression/zip/) 以及 [XML](/zh/web/xml/) 标准。 FODS 文件格式基于单个 XML 文档格式，也称为 Flat XML 或未压缩的 XML 文件。压缩格式的使用比基于 XML 的 OpenDocument 格式更常见，例如 [ODS](/zh/spreadsheet/ods/)。 OpenDocument 技术规范没有列出任何单一的 XML 文档，并且此类文档的更多详细信息不可用。
