## 什么是一 .accdc 文件？

扩展名为 .accdc 的文件是使用 Microsoft Access 创建的数据库包，其中包含数字签名的数据库。此类文件通常是通过签署现有 Microsoft Access 数据库文件（即 [ACCDB](/zh/database/accdb/)）以通过 Internet 发送给其他用户来创建的。这些文件的签名旨在确保最终用户收到的这些文件的可信度。 Microsoft Access 是一个关系数据库管理系统，用于创建和发布用于桌面和基于 Web 的内容管理系统的数据库。 ACCDC 文件可以使用 Microsoft Access 和 Office 365 打开。

## ACCDC 文件格式 - 更多信息

没有可供开发人员参考的 ACCDC 文件格式的公共规范。对样本 ACCDC 文件的简要分析表明，这些文件是二进制文件，因此没有进一步的详细信息。由于这些是从 ACCDB 文件生成的，因此这些 ACCDC 文件的内部结构类似于 ACCDB 文件。
