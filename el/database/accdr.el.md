## Τι είναι ένα αρχείο ACCDR;

Ένα αρχείο με επέκταση .accdr είναι ένα αρχείο βάσης δεδομένων Runtime που δημιουργήθηκε με βάση δεδομένων Microsoft Access και χρησιμοποιείται για τη δημιουργία σχεσιακών βάσεων δεδομένων. Δημιουργείται ως αρχείο βάσης δεδομένων μόνο για ανάγνωση που μπορεί να ανοίξει μόνο για προβολή και δεν μπορεί να επεξεργαστεί. Η Microsoft Access επιτρέπει επίσης τη δημιουργία άλλων τύπων αρχείων βάσης δεδομένων, όπως [ACCDB](/el/database/accdb/), [ACCDC](/el/database/accdc/) και [ACCDT](/el/database/accdt/). Τα αρχεία ACCDR στην πραγματικότητα είναι αρχεία ACCDB, αλλά δεν προβλέπεται προσθήκη, ενημέρωση ή διαγραφή δεδομένων από αυτά τα αρχεία. Τα αρχεία ACCDR μπορούν να ανοίξουν με τη Microsoft Access και το Office 365.

## Μορφή αρχείου ACCDR - Περισσότερες πληροφορίες

Τα αρχεία ACCDR είναι δυαδικά αρχεία που έχουν την ίδια δομή με τα αρχεία ACCDB αλλά δεν προβλέπεται ενημέρωση των περιεχομένων της βάσης δεδομένων. Από την έρευνα, είναι σαφές ότι η Microsoft εφαρμόζει ορισμένα ειδικά χαρακτηριστικά στο φάκελο Program Files που δεν επιτρέπουν την πραγματοποίηση αλλαγών σε αρχεία ACCDR. Εάν τα ίδια αρχεία ACCDR μετονομαστούν σε ACCDB, αυτά τα αρχεία γίνονται επεξεργάσιμα.
