## Какво е DISCOMAP файл?

DISCOMAP файлът е изходен файл, който се създава от Microsoft Web Services Discovery Tool, **Disco.exe**. Той съдържа резултатите, които са ASP.NET Web Services открити на конкретен URL адрес. Често срещани примери за такива открити услуги включват местоположения на .DISCO или .WSDL файлове. Когато инструментът Microsoft Web Discovery работи, той търси информация за откриване въз основа на XML съдържанието, намерено във файла DISCO. Резултатите се съхраняват във файла DISCOMAP, който може да се отвори с Microsoft Visual Studio IDE.
