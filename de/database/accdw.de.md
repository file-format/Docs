## Was ist eine ACCDW-Datei?

Eine Datei mit der Erweiterung .accdw ist eine mit Microsoft Access erstellte Verknüpfungsdatei und enthält Informationen über den Link zum Herunterladen einer Access-Datenbankdatei, dh [ACCDB](/de/database/accdb/) vom SharePoint-Server. Auf diese Weise können die Benutzer Datenbankdateien freigeben, die remote gehostet werden. Beim Öffnen der ACCDW-Datei wird die verknüpfte ACCDB-Datei als lokale Kopie auf den Computer des Benutzers heruntergeladen. Die heruntergeladene Datei wird am Standard-Download-Speicherort gespeichert und kann aufgerufen werden, indem Sie dorthin navigieren. Normalerweise werden diese Dateien heruntergeladen und unter dem Namen „SiteServer.accdb“ gespeichert, der sich auf die Client-Datenbanken bezieht.

## ACCDW-Dateiformat - Weitere Informationen

Die ACCDW-Datei ist eine XML-Datei, die einen Link zur SharePoint-Site bereitstellt, auf der die Access Services gehostet werden. Es ist nur eine Verknüpfung und erfordert eine Internetverbindung, um sie auszuführen. Im Falle von Online wird der SharePoint lokal zwischengespeichert, um die Möglichkeit zu bieten, ihn später offline zu nehmen.
