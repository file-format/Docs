## Was ist eine ACCDC-Datei?

Eine Datei mit der Erweiterung .accdc ist ein Datenbankpaket, das mit Microsoft Access erstellt wurde und eine digital signierte Datenbank enthält. Solche Dateien werden normalerweise erstellt, indem eine vorhandene Microsoft Access-Datenbankdatei (z. B. [ACCDB](/de/database/accdb/)) signiert wird, um sie über das Internet an andere Benutzer zu senden. Das Signieren dieser Dateien soll die Vertrauenswürdigkeit dieser Dateien sicherstellen, die von Endbenutzern empfangen werden. Microsoft Access ist ein relationales Datenbankverwaltungssystem, das zum Erstellen und Veröffentlichen von Datenbanken für die Verwendung in Desktop- und webbasierten Inhaltsverwaltungssystemen verwendet wird. ACCDC-Dateien können sowohl mit Microsoft Access als auch mit Office 365 geöffnet werden.

## ACCDC-Dateiformat – Weitere Informationen

Es sind keine öffentlichen Spezifikationen für das ACCDC-Dateiformat als Referenz für Entwickler verfügbar. Eine kurze Analyse von ACCDC-Beispieldateien zeigt, dass es sich bei Theses um Binärdateien handelt, für die keine weiteren Details verfügbar sind. Da diese aus den ACCDB-Dateien generiert werden, ist die interne Struktur dieser ACCDC-Dateien ähnlich wie bei ACCDB-Dateien.
