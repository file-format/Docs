## Was ist eine ADN-Datei?

Eine Datei mit der Erweiterung .adn ist eine leere Microsoft Access-Vorlagendatei, die zum Erstellen von Access-Projektdateien mit vorgespeicherten Einstellungen verwendet wird. Microsoft Access-Projektdateien werden als ADP-Dateien gespeichert und enthalten Informationen wie Datenbankverbindung und Projektstarteinstellungen. ADN-Dateien existieren für die Versionen 2007 und 2010 von Microsoft Access. Microsoft Access unterstützt andere Dateiformate wie [MDB](/de/database/mdb/), [ACCDB](/de/database/accdb/), [ACCDC](/de/database/accdc/), [ACCDE](/de/database/accde/ ) und viele weitere zusätzlich zu diesen ADN-Dateien. ADN-Dateien können mit Office 365 und Microsoft Access geöffnet werden.

## ADN-Dateiformat – Weitere Informationen

ADN sind Datenbankvorlagendateien, die als Binärdateien auf der Disc gespeichert werden. Die internen Dateiformatdetails von ADN-Dateien sind nicht als Referenz für Entwickler verfügbar.
