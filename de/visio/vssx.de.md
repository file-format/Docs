## Was ist eine VSSX-Datei?

Dateien mit der Erweiterung .vssx sind Zeichnungsschablonen, die mit [Microsoft Visio](https://products.office.com/en/visio/flowchart-software) 2013 und höher erstellt wurden. Das VSSX-Dateiformat kann mit Visio 2013 und höher geöffnet werden. Visio-Dateien sind für die Darstellung einer Vielzahl von Zeichnungselementen bekannt, z. B. eine Sammlung von Formen, Konnektoren, Flussdiagrammen, Netzwerklayouts, UML-Diagrammen, Softwarediagrammen, Datenbankmodellen, Objektzuordnungen und anderen ähnlichen Informationen. Microsoft Visio bietet auch die Möglichkeit, Visio-Dateien in verschiedene Dateiformate wie [PNG](/de/image/png/), [BMP](/de/image/bmp/), [PDF](/de/pdf/) und andere zu konvertieren. Es ist sowohl für Windows als auch für Mac OS verfügbar.

## VSSX-Dateiformat

Das VSSX-Dateiformat basiert auf dem seit 2007 von Microsoft eingeführten OpenOffice-Format. Es basiert auf dem [ZIP](/de/compression/zip/)-Archiv, das das gesamte Dateiformat basierend auf den XML-Dateiformatspezifikationen darstellt. Das binäre Dateiformatäquivalent von VSSX war VSS, das bis Visio 2007 unterstützt wurde. Sie können den Inhalt des VSSX-Dateiformats anzeigen, indem Sie seine Erweiterung durch .ZIP ersetzen und in einem beliebigen Archivierungsdateiformat wie WinZIP öffnen.

VSDX-Dateien basieren auf den Open Packaging Conventions und XML, und Entwickler können von diesem Format profitieren, indem sie lernen, wie sie programmgesteuert mit diesen Dateien arbeiten. Das Format erbt viele der gleichen XML-Strukturen wie seine Teile aus dem Visio-XML-Zeichnungsdateiformat (.vdx). Die Interoperabilität mit Visio-Dateien wird erheblich verbessert, da Software von Drittanbietern Visio-Dateien auf Dateiformatebene manipulieren kann.

Bestimmte andere Dateitypen, die das Dateiformat von Visio 2013 umfassen, umfassen:

* .vsdm (Visio-Makro-fähiges Zeichnen)
* .vssx (Visio-Schablone)
* .vssm (Visio-Makro-aktivierte Schablone)
* .vstx (Visio-Vorlage)
* .vstm (Visio-Makro-aktivierte Vorlage)

Jede Visio-Datei wird als Paket bezeichnet, das andere Dateien oder Teile enthält. Ein Paketteil kann eine XML-Datei, ein Bild oder sogar eine VBA-Lösung sein. Die Teile innerhalb des Pakets können in „Dokument“- und „Beziehungs“-Teile unterteilt werden.
