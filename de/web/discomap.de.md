## Was ist eine DISCOMAP-Datei?

Eine DISCOMAP-Datei ist eine Ausgabedatei, die vom Microsoft Web Services Discovery Tool **Disco.exe** erstellt wird. Es enthält die Ergebnisse, die von ASP.NET-Webdiensten unter einer bestimmten URL erkannt wurden. Häufige Beispiele für solche entdeckten Dienste sind Speicherorte von .DISCO- oder .WSDL-Dateien. Wenn das Microsoft Web Discovery-Tool ausgeführt wird, sucht es basierend auf dem in der DISCO-Datei gefundenen XML-Inhalt nach Erkennungsinformationen. Die Ergebnisse werden in der DISCOMAP-Datei gespeichert, die mit Microsoft Visual Studio IDE geöffnet werden kann.
