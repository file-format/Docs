## Qu'est-ce qu'un fichier ACCDR ?

Un fichier avec l'extension .accdr est un fichier de base de données Runtime créé avec la base de données Microsoft Access qui est utilisé pour créer des bases de données relationnelles. Il est créé en tant que fichier de base de données en lecture seule qui peut être ouvert pour la visualisation uniquement et ne peut pas être modifié. Microsoft Access permet également de créer d'autres types de fichiers de base de données tels que [ACCDB](/fr/database/accdb/), [ACCCDC](/fr/database/accdc/) et [ACCDT](/fr/database/accdt/). Les fichiers ACCDR sont en réalité des fichiers ACCDB, mais sans possibilité d'ajouter, de mettre à jour ou de supprimer des données de ces fichiers. Les fichiers ACCDR peuvent être ouverts avec Microsoft Access et Office 365.

## Format de fichier ACCDR - Plus d'informations

Les fichiers ACCDR sont des fichiers binaires qui ont la même structure que les fichiers ACCDB mais sans possibilité de mettre à jour le contenu de la base de données. D'après la recherche, il est clair que Microsoft applique certaines caractéristiques spéciales au dossier Program Files qui ne permettent pas d'apporter des modifications aux fichiers ACCDR. Si les mêmes fichiers ACCDR sont renommés en ACCDB, ces fichiers deviennent modifiables.
