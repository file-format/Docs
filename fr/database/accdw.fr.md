## Qu'est-ce qu'un fichier ACCDW ?

Un fichier avec l'extension .accdw est un fichier de lien créé avec Microsoft Access et contient des informations sur le lien pour télécharger un fichier de base de données Access, c'est-à-dire [ACCDB](/fr/database/accdb/) à partir du serveur SharePoint. Cela permet aux utilisateurs de partager des fichiers de base de données hébergés à distance. L'ouverture du fichier ACCDW télécharge le fichier ACCDB lié en tant que copie locale sur l'ordinateur de l'utilisateur. Le fichier téléchargé est enregistré dans l'emplacement de téléchargement par défaut et est accessible en y accédant. Généralement, ces fichiers sont téléchargés et stockés sous le nom "SiteServer.accdb" qui fait référence aux bases de données client.

## Format de fichier ACCDW - Plus d'informations

Le fichier ACCDW est un fichier XML qui fournit un lien vers le site SharePoint où les services d'accès sont hébergés. Il s'agit uniquement d'un raccourci et nécessite une connexion Internet pour les exécuter. En cas de connexion en ligne, le SharePoint est mis en cache localement afin de fournir la possibilité de le mettre hors ligne ultérieurement.
