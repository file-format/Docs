## Qu'est-ce qu'un fichier Oracle TRM ?

Un fichier TRM est un format de fichier de métadonnées utilisé par le système de gestion de base de données relationnelle Oracle 11g (RDBMS). Il est stocké avec les fichiers Oracle Trace ([.trc](/fr/database/trc/)) et contient des informations structurelles sur le fichier de trace. Le but du fichier TRM est de faciliter la recherche et la navigation dans les enregistrements à l'aide des informations de métadonnées. Le logiciel Oracle peut être utilisé pour ouvrir les fichiers TRM.

## Format de fichier TRM

Les fichiers TRM sont enregistrés dans le format de fichier propriétaire d'Oracle et les détails du format de fichier TRM ne sont pas disponibles publiquement. Un fichier TRC typique contient le SID du processus Oracle, le nom du processus et le numéro du processus du système d'exploitation. Les fichiers TRM contiennent des informations de métadonnées détaillées sur ces fichiers TRC pour la recherche et la navigation.
