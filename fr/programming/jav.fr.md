## Qu'est-ce qu'un fichier Jav ?

Un fichier Jav est un fichier de code source Java créé dans le langage de programmation Java. Il est similaire au fichier source Java et peut être compilé avec des interpréteurs Java. Comme les fichiers [Java](/fr/programming/java/), le fichier Jav est utilisé dans le développement d'applications Java. Il peut s'agir de l'un des nombreux fichiers de code source disponibles dans un projet Java.

## Format de fichier Java

Les fichiers Jav sont stockés sous forme de fichiers texte et peuvent être ouverts avec n'importe quel éditeur de texte. Les types de fichiers Java ont été couramment remplacés par le type de fichier Java et sont couramment utilisés pour écrire des applications Java. Les codes Java peuvent être écrits avec des IDE comme Eclipse et IntelliJ Idea, ce qui facilite le travail avec les fichiers Java.

### Exemple de format de fichier Java

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
