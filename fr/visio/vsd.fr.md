## Qu'est-ce qu'un fichier VSD ?

Les fichiers VSD sont des dessins créés avec l'application Microsoft Visio pour représenter une variété d'objets graphiques et l'interconnexion entre ceux-ci. Ces dessins peuvent contenir des objets visuels tels que des objets visuels, des organigrammes, des diagrammes UML, des flux d'informations, des organigrammes, des diagrammes de logiciels, la disposition du réseau, des modèles de base de données, des mappages d'objets et d'autres informations similaires. Microsoft Visio offre la possibilité de convertir des fichiers Visio dans un certain nombre de formats de fichiers différents, notamment [PNG](/fr/image/png/), [BMP](/fr/image/bmp/), [PDF](/fr/pdf/) et autres.

# Format de fichier VSD #

Le format de fichier VSD est le format de fichier initial introduit par Microsoft Visio pour stocker les éléments de dessin. La structure interne des fichiers VSD est basée sur le format de fichier binaire contrairement à [.VSDX](/fr/image/vsdx/) qui a été introduit avec Microsoft Visio 2013. Le nouveau format est basé sur le format de fichier Office Open XML et stocke les informations du projet dans un [ZIP](/fr/compression/zip/). Microsoft propose l'application vviewer.exe aux fichiers de dessin Visio au cas où Visio n'est pas installé sur le système.

## Bref historique ##

Visio a été initialement produit en tant que produit autonome par Shapeware Corporation en 1992 par sa version 1.0. Shapeware Corporation a changé le nom en Visio Corporation en 1995 pour profiter de la reconnaissance du marché. Microsoft renomme Vision en 2000 en tant qu'application Microsoft Office. Visio for Enterprise Architects a été introduit avec Visual Studio .NET en 2003 et 2005. Visio 2007 est sorti le 30 novembre 2006. Les rubans ont été introduits dans l'interface utilisateur par Microsoft Visio en 2010.
