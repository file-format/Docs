## Qu'est-ce qu'un fichier VSSX ?

Les fichiers avec l'extension .vssx sont des gabarits de dessin créés avec [Microsoft Visio](https://products.office.com/en/visio/flowchart-software) 2013 et versions ultérieures. Le format de fichier VSSX peut être ouvert avec Visio 2013 et supérieur. Les fichiers Visio sont connus pour la représentation d'une variété d'éléments de dessin tels que la collection de formes, les connecteurs, les organigrammes, la disposition du réseau, les diagrammes UML, les diagrammes de logiciels, les modèles de base de données, le mappage d'objets et d'autres informations similaires. Microsoft Visio offre également la possibilité de convertir des fichiers Visio en différents formats de fichiers tels que [PNG](/fr/image/png/), [BMP](/fr/image/bmp/), [PDF](/fr/pdf/) et autres. Il est disponible pour Windows et Mac OS.

## Format de fichier VSSX

Le format de fichier VSSX est basé sur le format OpenOffice adopté par Microsoft depuis 2007. Il est basé sur l'archive [ZIP](/fr/compression/zip/) qui représente le format de fichier global basé sur les spécifications de format de fichier XML. L'équivalent du format de fichier binaire de VSSX était VSS qui était pris en charge jusqu'à Visio 2007. Vous pouvez afficher le contenu du format de fichier VSSX en remplaçant son extension par .ZIP et l'ouvrir dans n'importe quel format de fichier d'archivage tel que WinZIP.

Les fichiers VSDX sont basés sur les conventions Open Packaging et XML et les développeurs peuvent bénéficier de ce format en apprenant à travailler avec ces fichiers par programmation. Le format hérite d'un grand nombre des mêmes structures XML que ses parties du format de fichier Visio XML Drawing (.vdx). L'interopérabilité avec les fichiers Visio est considérablement accrue car les logiciels tiers peuvent manipuler les fichiers Visio au niveau du format de fichier.

Certains autres types de fichiers qui composent le format de fichier Visio 2013 incluent :

* .vsdm (dessin compatible avec les macros Visio)
* .vssx (gabarit Visio)
* .vssm (gabarit prenant en charge les macros Visio)
* .vstx (modèle Visio)
* .vstm (modèle compatible avec les macros Visio)

Chaque fichier Visio est appelé package contenant d'autres fichiers ou parties. Une partie de package peut être un fichier XML, une image ou même une solution VBA. Les parties du package peuvent être divisées en parties "document" et "relation".
