## À propos de DOCX vers MHTML

Les pages Web peuvent être enregistrées dans un certain nombre de formats différents. MHTML est l'un d'entre eux où l'intégralité du contenu de la page est enregistrée dans un seul fichier d'archive contenant le code Web ainsi que les ressources. Microsoft Word vous permet d'exporter des fichiers DOCX vers MHTML en utilisant l'option Enregistrer sous. Mais si vous ne l'avez pas installé sur votre PC, vous pouvez utiliser des outils en ligne gratuits pour convertir DOCX en MHTML. L'application Aspose.Words est l'une de ces applications en ligne gratuites qui vous permet de convertir DOCX en MHTML en ligne.
