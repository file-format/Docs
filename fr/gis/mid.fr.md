## Qu'est-ce qu'un fichier MID ?

Un fichier MID dans la catégorie des fichiers SIG est un fichier de données SIG généré avec le logiciel d'analyse et de visualisation de données spatiales MapInfo. Il est généralement créé avec le fichier MapInfo Interchange Format ([MIF](/fr/gis/mif/)) correspondant et porte le même nom. Les fichiers MID stockent de grands ensembles de données contenant des informations d'attribut pour les objets de la carte et sont importés dans les attributs de l'objet.

## Format de fichier MID - Plus d'informations

Les fichiers MID sont enregistrés sur le disque au format de fichier binaire. Les attributs des fichiers MID sont enregistrés dans le même ordre que celui du fichier MIF correspondant. Par exemple, la troisième entité géométrique du fichier .mif possède les attributs stockés dans la troisième ligne du fichier .mid.
