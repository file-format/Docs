## Qu'est-ce qu'un fichier DISCOMAP ?

Un fichier DISCOMAP est un fichier de sortie créé par l'outil de découverte des services Web Microsoft, **Disco.exe**. Il contient les résultats qui sont des services Web ASP.NET découverts à une URL spécifique. Des exemples courants de ces services découverts incluent les emplacements de fichiers .DISCO ou .WSDL. Lorsque l'outil Microsoft Web Discovery s'exécute, il recherche des informations de découverte en fonction du contenu XML trouvé dans le fichier DISCO. Les résultats sont stockés dans le fichier DISCOMAP qui peut être ouvert avec Microsoft Visual Studio IDE.
