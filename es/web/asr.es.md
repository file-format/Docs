## ¿Qué es un archivo ASR?

Un archivo ASR es un archivo ActionScript creado con Adobe Dreamweaver que se carga y ejecuta en el lado del servidor. A menudo también lo crea el editor Contribute [HTML](/es/web/html/) que se suspendió hace mucho tiempo. El archivo ASR contiene secuencias de comandos escritas en el lenguaje de secuencias de comandos estándar y tiene la capacidad de acceder a funciones integradas basadas en servidor en ColdFusion.

## Formato de archivo ASR

El archivo ASR contiene una secuencia de comandos que se desarrolla en un lenguaje de secuencias de comandos estándar. Se puede abrir con Adobe Dreamweaver, Contribute HTML Editor y cualquier editor de texto como Notepad y Notepad++.
