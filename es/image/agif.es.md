## ¿Qué es un archivo AGIF?

Un archivo AGIF es un archivo GIF creado con Active GIF Creator. Es un archivo de proyecto que se puede abrir y ver en proyectos de Active GIF Creator. Cuando se abre en Active GIF Creator, muestra una secuencia de imágenes que finalmente conforman el archivo GIF finalizado con la extensión .gif.

## Formato de archivo AGIF

AGIF es un archivo de proyecto que se guarda como formato de archivo de imagen de trama. El archivo [GIF](/es/image/gif/) finalizado se puede abrir y mostrar como animación. Hay API de Python disponibles en línea que se pueden usar para crear archivos GIF animados.
