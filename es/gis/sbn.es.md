## ¿Qué es un archivo SBN?

Un archivo con extensión .sbn es un archivo de índice espacial que viene asociado con el archivo ESRI ArcGIS [.shp](/es/gis/shp/). Se utiliza para optimizar las consultas espaciales cuando se ha realizado un índice sobre los datos. El otro tipo de archivo que se usa junto con .sbn es el archivo .sbx. Los archivos SBN y SBX juntos forman un índice de forma para acelerar las consultas espaciales. Los archivos SBN son opcionales y se pueden abrir con [ESRI ArcGIS Pro](https://www.esri.com/en-us/arcgis/products/arcgis-pro/overview).

## Formato de archivo SBN: más información

Los archivos SBN se almacenan en el disco como archivos binarios y los detalles de su formato de archivo interno no están disponibles. Estos almacenan la representación binaria de un archivo SHP para consultas espaciales. El archivo de índice SBX se usa junto con los archivos SBN para acelerar las consultas espaciales en los archivos de forma.
