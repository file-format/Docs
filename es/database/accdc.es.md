## ¿Qué es un archivo ACCDC?

Un archivo con extensión .accdc es un paquete de base de datos creado con Microsoft Access y contiene una base de datos firmada digitalmente. Este tipo de archivos generalmente se crean mediante la firma de un archivo de base de datos de Microsoft Access existente (es decir, [ACCDB](/es/database/accdb/)) para enviarlos por Internet a otros usuarios. La firma de estos archivos está destinada a garantizar la confiabilidad de estos archivos recibidos por los usuarios finales. Microsoft Access es un sistema de administración de bases de datos relacionales que se utiliza para crear y publicar bases de datos para su uso en sistemas de administración de contenido de escritorio y basados en la web. Los archivos ACCDC se pueden abrir con Microsoft Access y Office 365.

## Formato de archivo ACCDC - Más información

No hay especificaciones públicas disponibles para el formato de archivo ACCDC para referencia del desarrollador. Un breve análisis de los archivos ACCDC de muestra muestra que estos son archivos binarios, sin más detalles disponibles como tales. Dado que estos se generan a partir de los archivos ACCDB, la estructura interna de estos archivos ACCDC es similar a los archivos ACCDB.
