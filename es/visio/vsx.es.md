## ¿Qué es un archivo VSX?

Los archivos con extensión .vsx se refieren a plantillas que consisten en dibujos y formas que se utilizan para crear diagramas en Microsoft Visio. Los archivos VSX se guardan en formato de archivo XML y fueron compatibles hasta Visio 2013. Estos son diferentes al formato de archivo principal [VSDX](/es/visio/vsdx/) que se introdujo con Microsoft Visio 2013. Los archivos VSX se pueden abrir en cualquier editor de texto para ver los contenidos. Los archivos VSX se pueden convertir a varios formatos de archivo diferentes, como PDF y HTML.

## Formato de archivo VSX ##

A continuación se muestra un archivo VSX de muestra abierto en el editor XML. Un archivo VSX típico tiene los siguientes nodos como se observa en su representación XML.


### Hojas de estilo ###

El nodo StyleSheets contiene información sobre las hojas de estilo utilizadas en el documento. Cada hoja de estilo puede tener información sobre propiedades de estilo, líneas, patrones de relleno, bloques de texto y otros elementos similares.

### Hoja de documento ###

El nodo DocumentSheet contiene información sobre las propiedades del documento y la información del nombre de usuario, y tiene el siguiente aspecto:

```
<DocProps>
<OutputFormat>0</OutputFormat>
<LockPreview>0</LockPreview>
<AddMarkup>0</AddMarkup>
<ViewMarkup>0</ViewMarkup>
<PreviewQuality>0</PreviewQuality>
<PreviewScope>0</PreviewScope>
<DocLangID>1033</DocLangID>
</DocProps>
<User NameU#"msvNoAutoConnect" ID#"1">
<Value>1</Value>
<Prompt F#"No Formula"/>
</User>
</DocumentSheet>
```
