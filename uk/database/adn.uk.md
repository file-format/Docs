## Що таке файл ADN?

Файл із розширенням .adn — це пустий файл шаблону Microsoft Access, який використовується для створення файлів проекту Access із попередньо збереженими налаштуваннями. Файли проекту Microsoft Access зберігаються як файли ADP і містять таку інформацію, як параметри підключення до бази даних і запуску проекту. Файли ADN існують для версій Microsoft Access 2007 і 2010. Microsoft Access підтримує інші формати файлів, як-от [MDB](/uk/database/mdb/), [ACCDB](/uk/database/accdb/), [ACCDC](/uk/database/accdc/), [ACCDE](/uk/database/accde/ ), і багато іншого на додаток до цих файлів ADN. Файли ADN можна відкривати за допомогою Office 365 і Microsoft Access.

## Формат файлу ADN - Додаткова інформація

ADN — це файли шаблонів бази даних, які зберігаються на диску як двійкові файли. Деталі внутрішнього формату файлів ADN недоступні для довідки розробників.
