## Що таке файл DTSX?

Файл із розширенням .dtsx (Data Transformation Services Package XML) — це файл Data Transformation Services (DTS), який використовується Microsoft SQL для посилання на кроки/правила міграції даних для перенесення даних з однієї бази даних до іншої. Це включає перетворення та будь-які додаткові етапи обробки, які можуть знадобитися під час передачі даних між початковою та кінцевою точками. Служби інтеграції SQL Server (SSIS), компонент Microsoft SQL Server, використовують файли DTSX для визначення етапів обробки даних між серверами баз даних. Файли DTSX можна відкривати за допомогою Microsoft SQL Server 2019.

## Формат файлу DTSX

Для переміщення даних між серверами баз даних потрібні правила й етапи обробки, щоб забезпечити цілісність даних під час цієї операції. SSIS гарантує, що всі ці дії з переміщення та узгодження даних з різних джерел на підприємстві відбуваються зручно. Саме тут з’являється DTSX, який визначає структури, які використовуватимуться SSIS для встановлення кроків, на яких дані можуть оброблятися під час виконання цих етапів.

DTSX базується на [XML](/uk/web/xml/) і задокументовано в [MS-DTSX](https://learn.microsoft.com/en-us/openspecs/sql_data_portability/ms-dtsx/235600e9-0c13-4b5b-a388-aa3c65aec1dd). Покращений рефакторинг DTSX XML — це DTSX 2.0, який включає нові атрибути структур, заміну іменованих властивостей як батьківських атрибутів XML, визначає значення за замовчуванням для більшості значень атрибутів і розміщення повторюваних елементів усередині батьківського елемента. Структури DTSX описані за допомогою цих [XML-схем](https://learn.microsoft.com/en-us/openspecs/sql_data_portability/ms-dtsx/e5095968-26ea-4824-a717-153ccee642dc), а структурний формат celar-text XML.
