## Що таке файл ACCDR?

Файл із розширенням .accdr — це файл бази даних Runtime, створений за допомогою бази даних Microsoft Access, яка використовується для створення реляційних баз даних. Він створюється як файл бази даних лише для читання, який можна відкрити лише для перегляду та не можна редагувати. Microsoft Access також дозволяє створювати інші типи файлів бази даних, наприклад [ACCDB](/uk/database/accdb/), [ACCDC](/uk/database/accdc/) і [ACCDT](/uk/database/accdt/). Файли ACCDR насправді є файлами ACCDB, але не мають можливості додавати, оновлювати або видаляти дані з цих файлів. Файли ACCDR можна відкривати за допомогою Microsoft Access і Office 365.

## Формат файлу ACCDR - Додаткова інформація

Файли ACCDR — це двійкові файли, які мають ту саму структуру, що й файли ACCDB, але не мають можливості оновлювати вміст бази даних. З досліджень стало зрозуміло, що корпорація Майкрософт застосовує деякі спеціальні характеристики до папки Program Files, які не дозволяють вносити зміни до файлів ACCDR. Якщо ті самі файли ACCDR перейменувати на ACCDB, ці файли стають доступними для редагування.
