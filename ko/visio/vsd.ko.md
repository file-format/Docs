## .VSD 파일이란?

VSD 파일은 다양한 그래픽 개체와 이들 간의 상호 연결을 나타내기 위해 Microsoft Visio 응용 프로그램으로 만든 도면입니다. 이러한 도면에는 시각적 개체, 순서도, UML 다이어그램, 정보 흐름, 조직도, 소프트웨어 다이어그램, 네트워크 레이아웃, 데이터베이스 모델, 개체 매핑 및 기타 유사한 정보와 같은 시각적 개체가 포함될 수 있습니다. Microsoft Visio는 Visio 파일을 [PNG](/ko/image/png/), [BMP](/ko/image/bmp/), [PDF](/ko/pdf/) 등을 포함한 다양한 파일 형식으로 변환하는 기능을 제공합니다.

# VSD 파일 형식 #

VSD 파일 형식은 드로잉 요소를 저장하기 위해 Microsoft Visio에서 도입한 초기 파일 형식입니다. VSD 파일의 내부 구조는 Microsoft Visio 2013에서 도입된 [.VSDX](/ko/image/vsdx/)와 달리 바이너리 파일 형식을 기반으로 합니다. 새로운 형식은 Office Open XML 파일 형식을 기반으로 하며 프로젝트 정보를 [ZIP](/ko/compression/zip/) 아카이브. Microsoft는 시스템에 Visio가 설치되어 있지 않은 경우 Visio 도면 파일에 vviewer.exe 응용 프로그램을 제공합니다.

## 간략한 역사 ##

Visio는 1992년 Shapeware Corporation에서 버전 1.0으로 독립 실행형 제품으로 생산했습니다. Shapeware Corporation은 시장 인지도를 활용하기 위해 1995년에 Visio Corporation으로 이름을 변경했습니다. Microsoft는 2000년에 Vision을 Microsoft Office 응용 프로그램으로 다시 브랜드화했습니다. Enterprise Architects용 Visio는 2003년 및 2005년에 Visual Studio .NET과 함께 도입되었습니다. Visio 2007은 2006년 11월 30일에 릴리스되었습니다. 리본은 2010년 Microsoft Visio에서 사용자 인터페이스에 도입되었습니다.
