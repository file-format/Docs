## Vad är en DISCOMAP fil?

En DISCOMAP-fil är en utdatafil som skapas av Microsoft Web Services Discovery Tool, **Disco.exe**. Den innehåller resultaten som är ASP.NET Web Services som upptäckts på en specifik URL. Vanliga exempel på sådana upptäckta tjänster inkluderar platser för en .DISCO- eller .WSDL-filer. När Microsoft Web Discovery-verktyget körs söker det efter information baserat på XML-innehållet som finns i DISCO-filen. Resultaten lagras i DISCOMAP-filen som kan öppnas med Microsoft Visual Studio IDE.
