## Vad är en ACCDR fil?

En fil med tillägget .accdr är en Runtime-databasfil skapad med Microsoft Access-databas som används för att skapa relationsdatabaser. Den skapas som en skrivskyddad databasfil som endast kan öppnas för visning och inte kan redigeras. Microsoft Access tillåter också att skapa andra typer av databasfiler som [ACCDB](/sv/database/accdb/), [ACCDC](/sv/database/accdc/) och [ACCDT](/sv/database/accdt/). ACCDR-filer är i verkligheten ACCDB-filer, men utan möjlighet att lägga till, uppdatera eller ta bort data från dessa filer. ACCDR-filer kan öppnas med Microsoft Access och Office 365.

## ACCDR-filformat - Mer information

ACCDR-filer är binära filer som har samma struktur som ACCDB-filerna men utan bestämmelser för att uppdatera innehållet i databasen. Från forskning är det tydligt att Microsoft tillämpar vissa speciella egenskaper på mappen Program Files som inte tillåter ändringar att göras i ACCDR-filer. Om samma ACCDR-filer döps om till ACCDB, blir dessa filer redigerbara.
