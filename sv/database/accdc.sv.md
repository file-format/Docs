## Vad är en ACCDC fil?

En fil med filtillägget .accdc är ett databaspaket skapat med Microsoft Access och innehåller en digitalt signerad databas. Sådana typer av filer skapas vanligtvis genom att signera en befintlig Microsoft Access Database-fil (dvs [ACCDB](/sv/database/accdb/)) för att skicka över internet till andra användare. Signeringen av dessa filer är avsedd att säkerställa förtroendevärdigheten hos dessa filer som tas emot av slutanvändare. Microsoft Access är ett relationellt databashanteringssystem som används för att skapa och publicera databaser för användning i såväl stationära som webbaserade innehållshanteringssystem. ACCDC-filer kan öppnas med Microsoft Access såväl som Office 365.
