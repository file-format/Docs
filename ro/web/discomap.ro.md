## Ce este un fișier DISCOMAP?

Un fișier DISCOMAP este un fișier de ieșire care este creat de Microsoft Web Services Discovery Tool, **Disco.exe**. Conține rezultatele care sunt ASP.NET Web Services descoperite la adresa URL specifică. Exemple comune de astfel de servicii descoperite includ locațiile unui fișier .DISCO sau .WSDL. Când rulează instrumentul Microsoft Web Discovery, acesta caută informații de descoperire pe baza conținutului XML găsit în fișierul DISCO. Rezultatele sunt stocate în fișierul DISCOMAP care poate fi deschis cu Microsoft Visual Studio IDE.
