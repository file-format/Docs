## Ce este un fișier ACCDW?

Un fișier cu extensia .accdw este un fișier de legătură creat cu Microsoft Access și conține informații despre legătura pentru descărcarea unui fișier de bază de date Access, adică [ACCDB](/ro/database/accdb/) de pe serverul SharePoint. Acest lucru permite utilizatorilor să partajeze fișierele bazei de date care sunt găzduite de la distanță. Deschiderea fișierului ACCDW descarcă fișierul ACCDB legat ca o copie locală pe computerul utilizatorului. Fișierul descărcat este salvat în locația implicită de descărcare și poate fi accesat navigând la acesta. De obicei, aceste fișiere sunt descărcate și stocate cu numele „SiteServer.accdb” care se referă la bazele de date client.

## Format de fișier ACCDW - Mai multe informații

Fișierul ACCDW este un fișier XML care oferă un link către site-ul SharePoint unde sunt găzduite serviciile de acces. Este doar o comandă rapidă și necesită conexiune la internet pentru a le rula. În cazul online, SharePoint este stocat în cache local pentru a oferi opțiunea de a-l deconecta mai târziu.
