## Ce este un fișier VSD?

Fișierele VSD sunt desene create cu aplicația Microsoft Visio pentru a reprezenta varietatea de obiecte grafice și interconexiunea dintre acestea. Astfel de desene pot conține obiecte vizuale, cum ar fi obiecte vizuale, diagrame de flux, diagramă UML, flux de informații, organigrame, diagrame software, aspect de rețea, modele de baze de date, cartografiere de obiecte și alte informații similare. Microsoft Visio oferă capacitatea de a converti fișiere Visio într-un număr de formate de fișiere diferite, inclusiv [PNG](/ro/image/png/), [BMP](/ro/image/bmp/), [PDF](/ro/pdf/) și altele.

# Format fișier VSD #

Formatul de fișier VSD este formatul de fișier inițial introdus de Microsoft Visio pentru stocarea elementelor de desen. Structura internă a fișierelor VSD se bazează pe formatul de fișier binar, spre deosebire de [.VSDX](/ro/image/vsdx/) care a fost introdus cu Microsoft Visio 2013. Noul format se bazează pe formatul de fișier Office Open XML și stochează informații despre proiect într-un [ZIP](/ro/compression/zip/) arhivă. Microsoft oferă aplicația vviewer.exe pentru desenele Visio în cazul în care Visio nu este instalat pe sistem.

## Scurt istoric ##

Visio a fost produs inițial ca produs independent de Shapeware Corporation în 1992, prin versiunea sa 1.0. Shapeware Corporation și-a schimbat numele în Visio Corporation în 1995 pentru a profita de recunoașterea pieței. Microsoft a redenumit Vision în 2000 ca aplicație Microsoft Office. Visio for Enterprise Architects a fost introdus cu Visual Studio .NET în 2003 și 2005. Visio 2007 a fost lansat pe 30 noiembrie 2006. Panglicile au fost introduse în interfața cu utilizatorul de către Microsoft Visio în 2010.
