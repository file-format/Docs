## Ce este un fișier SBN?

Un fișier cu extensia .sbn este un fișier index spațial care vine asociat cu fișierul ESRI ArcGIS [.shp](/ro/gis/shp/). Este folosit pentru optimizarea interogărilor spațiale atunci când a fost efectuat un index asupra datelor. Celălalt tip de fișier folosit alături de .sbn este fișierul .sbx. Fișierele SBN și SBX formează împreună un index de formă pentru a accelera interogările spațiale. Fișierele SBN sunt opționale și pot fi deschise cu [ESRI ArcGIS Pro](https://www.esri.com/en-us/arcgis/products/arcgis-pro/overview).

## Format de fișier SBN - Mai multe informații

Fișierele SBN sunt stocate pe disc ca fișiere binare, iar detaliile lor interne ale formatului de fișier nu sunt disponibile. Acestea stochează reprezentarea binară a unui fișier SHP pentru interogări spațiale. Fișierul index SBX este utilizat alături de fișierele SBN pentru a accelera interogările spațiale pe fișierele shape.
