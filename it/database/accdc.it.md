## Che cos'è un file ACCDC?

Un file con estensione .accdc è un pacchetto di database creato con Microsoft Access e contiene un database con firma digitale. Questo tipo di file viene solitamente creato firmando un file di database di Microsoft Access esistente (ad esempio [ACCDB](/it/database/accdb/)) per l'invio su Internet ad altri utenti. La firma di questi file ha lo scopo di garantire l'affidabilità di questi file ricevuti dagli utenti finali. Microsoft Access è un sistema di gestione di database relazionale utilizzato per creare e pubblicare database da utilizzare nei sistemi di gestione dei contenuti desktop e basati sul Web. I file ACCDC possono essere aperti con Microsoft Access e Office 365.

## Formato file ACCDC - Ulteriori informazioni

Non sono disponibili specifiche pubbliche per il formato di file ACCDC per riferimento dello sviluppatore. Una breve analisi dei file ACCDC di esempio mostra che si tratta di file binari, senza ulteriori dettagli disponibili in quanto tali. Poiché questi sono generati dai file ACCDB, la struttura interna di questi file ACCDC è simile ai file ACCDB.
