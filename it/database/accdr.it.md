## Che cos'è un file ACCDR?

Un file con estensione .accdr è un file di database di runtime creato con il database di Microsoft Access che viene utilizzato per creare database relazionali. Viene creato come file di database di sola lettura che può essere aperto solo per la visualizzazione e non può essere modificato. Microsoft Access consente anche di creare altri tipi di file di database come [ACCDB](/it/database/accdb/), [ACCDC](/it/database/accdc/) e [ACCDT](/it/database/accdt/). I file ACCDR in realtà sono file ACCDB, ma senza alcuna disposizione per aggiungere, aggiornare o eliminare dati da questi file. I file ACCDR possono essere aperti con Microsoft Access e Office 365.

## Formato file ACCDR - Ulteriori informazioni

I file ACCDR sono file binari che hanno la stessa struttura dei file ACCDB ma senza alcuna disposizione per aggiornare il contenuto del database. Dalla ricerca, è chiaro che Microsoft applica alcune caratteristiche speciali alla cartella Programmi che non consentono di apportare modifiche ai file ACCDR. Se gli stessi file ACCDR vengono rinominati in ACCDB, questi file diventano modificabili.
