## Che cos'è un file ACCDW?

Un file con estensione .accdw è un file di collegamento creato con Microsoft Access e contiene informazioni sul collegamento per scaricare un file di database di Access, ad esempio [ACCDB](/it/database/accdb/) dal server SharePoint. Ciò consente agli utenti di condividere file di database ospitati in remoto. L'apertura del file ACCDW scarica il file ACCDB collegato come copia locale sul computer dell'utente. Il file scaricato viene salvato nel percorso di download predefinito ed è possibile accedervi navigando su di esso. Solitamente questi file vengono scaricati e archiviati con il nome "SiteServer.accdb" che fa riferimento ai database dei client.

## Formato file ACCDW - Ulteriori informazioni

Il file ACCDW è un file XML che fornisce un collegamento al sito di SharePoint in cui sono ospitati i servizi di accesso. È solo una scorciatoia e richiede una connessione a Internet per eseguirli. In caso di online, SharePoint viene memorizzato nella cache in locale per fornire l'opzione per portarlo offline in un secondo momento.
