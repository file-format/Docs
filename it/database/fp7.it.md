## Che cos'è un file FP7?

Un file con estensione .fp7 è un file di database creato con FileMaker Pro versioni 7-11. Memorizza i dati in tabelle con record in cui ogni record contiene uno o più campi. I file FP7 hanno la capacità di memorizzare dati di grandi dimensioni, inclusi dati di database personali e record organizzativi. Una versione più recente di FileMaker Pro è [FMP12](/it/database/fmp12/), introdotta con il lancio di FileMake Pro versione 12. Oltre ai dati testuali, il file FP7 può contenere anche moduli, grafici e altri oggetti di database . I file FP7 possono essere aperti con FileMaker Pro Advanced.

## Formato file 7PQ

I file FP7 vengono salvati su disco come file binari nel formato file proprietario di FileMaker. I dettagli del formato file interno del formato file FP7 non sono disponibili pubblicamente.
