## Che cos'è un file CSP?

Un file CSP è un file di pagina Web dinamico che contiene contenuto scritto per essere eseguito su RadGs Concept Application Server. Viene compilato ed eseguito lato server, dopodiché viene servito al browser dell'utente richiedente tramite protocollo HTTP. A differenza del formato di file CON, i file CSP non richiedono l'installazione del client Concept presso l'utente per la visualizzazione di questi file. È possibile visualizzare file CSP simili a qualsiasi altra pagina lato server come [PHP](/it/programming/php/) e [ASP](/it/web/asp/).
