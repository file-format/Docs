## Che cos'è un file LRC?

Un file LRC è un file di testi basato su testo che contiene i testi di una canzone audio. Quando la canzone audio viene riprodotta con un lettore musicale o un software di riproduzione audio su un computer, il file LRC viene letto in parallelo e visualizzato con il tempo. I file LRC contengono cue point per la visualizzazione dei testi durante la riproduzione del brano. In generale, i file LRC hanno lo stesso nome del file audio corrispondente come audio.mp3 e audio.lrc. I file LRC sono simili ai file dei sottotitoli come [SRT](/it/video/srt/).

## Formato file LRC - Ulteriori informazioni

I file in formato testo LRC vengono salvati come file di testo normale e possono essere aperti e modificati in un file di testo. Il contenuto di un file LRC contiene informazioni sul colore per la visualizzazione del testo dei testi.

## Formati LRC

Esistono diversi formati di file LRC che sono stati sviluppati nel tempo. Queste

### Formato LRC semplice

I tag del tempo di riga sono nel formato [mm:ss.xx] dove mm sono i minuti, ss sono i secondi e xx sono i centesimi di secondo.

```
[00:12.00]Line 1 lyrics
[00:17.20]Line 2 lyrics
[00:21.10]Line 3 lyrics
...
mm:ss.xxlast lyrics line
```

### Formato semplice esteso

Include la possibilità di cambiare e specificare il genere dei testi usando M: Male, F: Female, D: Duet.

```
[00:12.00]Line 1 lyrics
[00:17.20]F: Line 2 lyrics
[00:21.10]M: Line 3 lyrics
[00:24.00]Line 4 lyrics
[00:28.25]D: Line 5 lyrics
[00:29.02]Line 6 lyrics
```
### Formato LRC avanzato

Il formato LRC avanzato è una versione estesa del formato LRC semplice. Aggiunge un tag Word Time nel formato<mm:ss.xx> alla fine della parola precedente.

```
[ar: Jade Michael]
[al: Sarah Hi]
[au: Jade Michael]
[length: 2:58]
[by: lrc-maker]
[ti: Somebody to Love]

[00:00.00] <00:00.04> When <00:00.16> the <00:00.82> truth <00:01.29> is <00:01.63> found <00:03.09> to <00:03.37> be <00:05.92> lies
[00:06.47] <00:07.67> And <00:07.94> all <00:08.36> the <00:08.63> joy <00:10.28> within <00:10.53> you <00:13.09> dies
[00:13.34] <00:14.32> Don't <00:14.73> you <00:15.14> want <00:15.57> somebody <00:16.09> to <00:16.46> love
```
