## Che cos'è un file APPXUPLOAD?

Un file APPXUPLOAD è un singolo file di pacchetto che contiene più pacchetti di app o app bundle per il caricamento su Microsoft Store. Viene generato automaticamente da [Microsoft Visual Studio](https://visualstudio.microsoft.com/) quando gli sviluppatori creano pacchetti dell'app per la pubblicazione su [Microsoft Store](https://www.microsoft.com/en-us/ negozio/app/finestre). Oltre a un file [APPX](/it/programming/appx/), il file APPXUPLOAD contiene un file di simboli che analizza le prestazioni dell'app dopo che è stata pubblicata in Microsoft Store.
