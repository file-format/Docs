## Che cos'è un file ML?

Un file ML è un file di testo creato con il linguaggio di programmazione funzionale generico ML (spesso noto come Meta Language). Contiene codice sorgente ML che è più spesso utilizzato nella ricerca e nella progettazione di linguaggi di programmazione utilizzando una semantica completamente definita. Esistono più varianti di ML con Caml come quella ampiamente utilizzata. Altri includono Objective Caml (OCaml), JoCaml, Extended ML e MacroML. Il linguaggio ML è comunemente usato per la scrittura di compilatori, la dimostrazione automatizzata di teoremi e le verifiche formali.
