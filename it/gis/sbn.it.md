## Che cos'è un file SBN?

Un file con estensione .sbn è un file di indice spaziale associato al file ESRI ArcGIS [.shp](/it/gis/shp/). Viene utilizzato per ottimizzare le query spaziali quando è stato eseguito un indice sui dati. L'altro tipo di file utilizzato insieme a .sbn è il file .sbx. I file SBN e SBX insieme costituiscono un indice di forma per velocizzare le query spaziali. I file SBN sono opzionali e possono essere aperti con [ESRI ArcGIS Pro](https://www.esri.com/en-us/arcgis/products/arcgis-pro/overview).

## Formato file SBN - Ulteriori informazioni

I file SBN vengono archiviati su disco come file binari e i dettagli sul formato file interno non sono disponibili. Questi memorizzano la rappresentazione binaria di un file SHP per le query spaziali. Il file di indice SBX viene utilizzato insieme ai file SBN per velocizzare le query spaziali sugli shapefile.
