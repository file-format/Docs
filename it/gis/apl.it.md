## Che cos'è un file APL?

Un file con estensione .apl è un file di livello mappa utilizzato dal software applicativo ESRI ArcPad. Contiene informazioni su un sottoinsieme di proprietà del layer della mappa per una mappa ArcGIS esportata. Viene utilizzato dal personale sul campo per acquisire, analizzare e visualizzare informazioni geografiche sotto forma di immagini raster e vettoriali. Include la simbologia per la visualizzazione di caratteristiche, scale ed espressioni di etichetta per il livello della mappa. I file APL possono essere aperti con ArcPad Pro su dispositivi portatili utilizzati nel lavoro sul campo.

## Formato file APL

Gli APL sono file GIS binari e le informazioni sul formato del file non sono disponibili pubblicamente.
