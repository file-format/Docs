## Che cos'è un file TXT?

Un file con estensione .TXT rappresenta un documento di testo che contiene testo normale sotto forma di righe. I paragrafi in un documento di testo sono riconosciuti dai ritorni a capo e sono utilizzati per una migliore disposizione del contenuto del file. Un documento di testo standard può essere aperto in qualsiasi editor di testo o applicazione di elaborazione testi su diversi sistemi operativi. Tutto il testo contenuto in tale file è in formato leggibile e rappresentato da una sequenza di caratteri.

I file di testo possono memorizzare grandi quantità di dati in quanto non vi sono limitazioni alla dimensione dei contenuti. Tuttavia, gli editor di testo che aprono file così grandi devono essere intelligenti per caricarli e visualizzarli. Quasi tutti i sistemi operativi sono dotati di editor di testo che consentono di creare e modificare file di testo. Ad esempio, il sistema operativo Windows viene fornito con Blocco note e Wordpad per questo scopo. Allo stesso modo, MacOS viene fornito con TextEdit per la creazione e la modifica di documenti di testo. Ci sono, tuttavia, anche altri editor di testo gratuiti disponibili su Internet che ti offrono la possibilità di lavorare con documenti di testo come Notepad ++ che è molto più avanzato in termini di funzionalità.

## Specifiche del formato file ##

Il formato del file di testo non ha specifiche speciali per il formato del file. I file di testo hanno il tipo MIME "testo/normale" e hanno una formattazione minima o nulla. Ciò consente agli editor di testo di aprire tali file senza altri requisiti. Il set di caratteri predefinito dei file di testo è ASCII utilizzato per la creazione e la visualizzazione del contenuto dei file di testo. I caratteri vengono codificati utilizzando il set di caratteri ASCII, ma ciò impone limitazioni all'utilizzo di caratteri come il simbolo della sterlina, del dollaro e dell'euro che non possono essere rappresentati utilizzando il set di caratteri ASCII. Pertanto, i file di testo possono anche essere salvati in formato Unicode, con UTF-8 che è il più utilizzato.

### Formato file di testo di Windows ###

I file di testo sul sistema operativo Windows sono costituiti da diverse righe in cui ogni riga è composta da una sequenza di caratteri. Ogni riga implicita dell'utente è definita dalla combinazione di due caratteri, ad esempio il ritorno a capo (CR) e l'avanzamento riga (LF). I file di testo di Windows possono essere con codifica ANSI, OEM, Unicode o UTF-8. La codifica UTF-16 aiuta a salvare le informazioni in un file di testo che richiede due byte per la rappresentazione. Tali file di solito iniziano con Byte Order Mark (BOM) che comunica l'endianness del contenuto del file. Va notato che altre applicazioni sul sistema operativo Windows possono memorizzare informazioni in formato file di testo ma con estensioni di file diverse per rappresentare il testo specifico dell'applicazione. Ad esempio, i linguaggi di programmazione di solito salvano il codice in un file di testo ma con le proprie estensioni.

### Formato file di testo Unix ###

Tutti questi sistemi definiscono un file di testo come un file i cui caratteri sono organizzati in zero o più righe. Ogni riga è una sequenza di zero o più caratteri non di nuova riga e un carattere di fine riga di fine, normalmente LF.
