## Che cos'è un file ORB? ##

Il formato file ORB appartiene al tipo di file ebooks che può essere letto utilizzando ORB Reader, un software antico particolarmente utilizzato per leggere o modificare i file ebook con estensione .orb. Si prega di notare che ORB sta per Original Book Reader e il formato è stato sviluppato da Original eBook. Pertanto, nessuna o pochissime informazioni sono disponibili sugli sviluppatori di ORB su Internet, il che è un segno che ORB Reader potrebbe essere stato interrotto o non più disponibile nell'attuale mercato Internet. Quindi, alcune persone potrebbero sentirsi tristi se hanno dati di dimensioni maggiori, ma non hanno opzioni sufficienti per fare qualcosa con esso.

## Struttura del file ORB ##

I file con estensione .orb sono in realtà compressi con compressione Zip; contenente uno o più file [SWF](/it/page-description-language/swf/) (File Macromedia Flash) per creare un eBook. Potrebbe sorgere una domanda nella tua mente: come possiamo aprire o leggere questo formato di file? La risposta è che puoi aprirlo con Original Book Reader, noto anche come **ORB Reader**. ORB Reader sembra essere fuori produzione o non più disponibile su Internet, ma in passato era disponibile per Microsoft Windows e macOS.

## Visualizza un file ORB ##

Poiché l'ORB Reader veniva utilizzato ai vecchi tempi; potrebbe essere possibile che tu abbia alcuni dati importanti sotto forma di file ORB. Ora, come trucco, puoi farlo in modo manuale. Come accennato in precedenza, i file ORB sono stati compressi con [ZIP](/it/compression/zip/) Compression; puoi rinominare l'estensione ".orb" in ".zip" e quindi saresti in grado di estrarre l'intero contenuto di un file ORB.
