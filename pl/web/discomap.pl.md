## Czym jest plik DISCOMAP?

Plik DISCOMAP to plik wyjściowy tworzony przez narzędzie Microsoft Web Services Discovery Tool, **Disco.exe**. Zawiera wyniki, które są usługami sieci Web ASP.NET wykrytymi pod określonym adresem URL. Typowymi przykładami takich wykrytych usług są lokalizacje plików .DISCO lub .WSDL. Gdy narzędzie Microsoft Web Discovery jest uruchomione, wyszukuje informacje na podstawie zawartości XML znalezionej w pliku DISCO. Wyniki są przechowywane w pliku DISCOMAP, który można otworzyć za pomocą Microsoft Visual Studio IDE.
