## O que é um arquivo ACCDW?

Um arquivo com extensão .accdw é um arquivo de link criado com o Microsoft Access e contém informações sobre o link para baixar um arquivo de banco de dados do Access, ou seja, [ACCDB](/pt/database/accdb/) do servidor SharePoint. Isso permite que os usuários compartilhem arquivos de banco de dados hospedados remotamente. A abertura do arquivo ACCDW baixa o arquivo ACCDB vinculado como uma cópia local no computador do usuário. O arquivo baixado é salvo no local de download padrão e pode ser acessado navegando até ele. Normalmente, esses arquivos são baixados e armazenados com o nome "SiteServer.accdb" que se refere aos bancos de dados do cliente.

## Formato de arquivo ACCDW - Mais informações

O arquivo ACCDW é um arquivo XML que fornece um link para o site do SharePoint onde os Serviços do Access estão hospedados. É apenas um atalho e requer conexão com a Internet para executá-los. No caso de online, o SharePoint é armazenado em cache localmente para fornecer a opção de colocá-lo offline posteriormente.
