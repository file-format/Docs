## O que é um arquivo ACCDC?

Um arquivo com extensão .accdc é um pacote de banco de dados criado com o Microsoft Access e contém um banco de dados assinado digitalmente. Esses tipos de arquivos geralmente são criados assinando um arquivo existente do Microsoft Access Database (ou seja, [ACCDB](/pt/database/accdb/)) para envio pela Internet para outros usuários. A assinatura desses arquivos destina-se a garantir a confiabilidade desses arquivos recebidos pelos usuários finais. O Microsoft Access é um sistema de gerenciamento de banco de dados relacional que é usado para criar e publicar bancos de dados para uso em desktop, bem como em sistemas de gerenciamento de conteúdo baseados na web. Os arquivos ACCDC podem ser abertos com o Microsoft Access e com o Office 365.

## Formato de arquivo ACCDC - Mais informações

Não há especificações públicas disponíveis para o formato de arquivo ACCDC para referência do desenvolvedor. Uma breve análise de arquivos ACCDC de amostra mostra que as teses são arquivos binários, sem mais detalhes disponíveis como tal. Como eles são gerados a partir dos arquivos ACCDB, a estrutura interna desses arquivos ACCDC é semelhante aos arquivos ACCDB.
