## O que é um arquivo ACCDR?

Um arquivo com extensão .accdr é um arquivo de banco de dados Runtime criado com o banco de dados Microsoft Access que é usado para criar bancos de dados relacionais. Ele é criado como um arquivo de banco de dados somente leitura que pode ser aberto apenas para visualização e não pode ser editado. O Microsoft Access também permite criar outros tipos de arquivos de banco de dados, como [ACCDB](/pt/database/accdb/), [ACCDC](/pt/database/accdc/) e [ACCDT](/pt/database/accdt/). Os arquivos ACCDR na realidade são arquivos ACCDB, mas sem provisão para adicionar, atualizar ou excluir dados desses arquivos. Os arquivos ACCDR podem ser abertos com o Microsoft Access e o Office 365.

## Formato de arquivo ACCDR - Mais informações

Os arquivos ACCDR são arquivos binários que têm a mesma estrutura dos arquivos ACCDB, mas sem provisão para atualizar o conteúdo do banco de dados. Pela pesquisa, fica claro que a Microsoft aplica algumas características especiais à pasta Arquivos de Programas que não permitem que sejam feitas alterações nos arquivos ACCDR. Se os mesmos arquivos ACCDR forem renomeados para ACCDB, esses arquivos se tornarão editáveis.
