## O que é um arquivo FP7?

Um arquivo com extensão .fp7 é um arquivo de banco de dados criado com o FileMaker Pro versões 7-11. Armazena os dados em tabelas com registros onde cada registro contém um ou mais campos. Os arquivos FP7 têm a capacidade de armazenar grandes dados, incluindo dados de bancos de dados pessoais e registros organizacionais. Uma versão mais recente do FileMaker Pro é [FMP12](/pt/database/fmp12/) que foi introduzida com o lançamento do FileMake Pro versão 12. Além dos dados textuais, o arquivo FP7 também pode conter formulários, gráficos e outros objetos de banco de dados . Os arquivos FP7 podem ser abertos com o FileMaker Pro Advanced.

## Formato de arquivo FP7

Os arquivos FP7 são salvos em disco como arquivos binários no formato de arquivo proprietário do FileMaker. Os detalhes do formato de arquivo interno do formato de arquivo FP7 não estão disponíveis publicamente.
