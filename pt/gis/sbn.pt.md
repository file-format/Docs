## O que é um arquivo SBN?

Um arquivo com extensão .sbn é um arquivo de índice espacial que vem associado ao arquivo ESRI ArcGIS [.shp](/pt/gis/shp/). Ele é usado para otimizar consultas espaciais quando um índice foi executado nos dados. O outro tipo de arquivo usado junto com .sbn é o arquivo .sbx. Os arquivos SBN e SBX juntos formam um índice de forma para acelerar as consultas espaciais. Os arquivos SBN são opcionais e podem ser abertos com o [ESRI ArcGIS Pro](https://www.esri.com/en-us/arcgis/products/arcgis-pro/overview).

## Formato de arquivo SBN - Mais informações

Os arquivos SBN são armazenados em disco como arquivos binários e seus detalhes de formato de arquivo interno não estão disponíveis. Eles armazenam a representação binária de um arquivo SHP para consultas espaciais. O arquivo de índice SBX é usado junto com os arquivos SBN para acelerar as consultas espaciais em shapefiles.
