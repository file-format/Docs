## O que é um arquivo DEX?

Um arquivo DEX é um arquivo de planilha criado com o Microsoft Excel. É semelhante aos arquivos [XLSX](/pt/spreadsheet/xlsx/), mas está no formato de arquivo binário. Os dados são armazenados em linhas e colunas em um arquivo DEX. Embora os arquivos DEX também sejam do Microsoft Excel, eles são menos populares. Em comparação, o formato de arquivo XLSX é a opção padrão para salvar pastas de trabalho do Excel. Os arquivos DEX foram marcados como obsoletos e não são mais suportados pelo Microsoft Excel.
