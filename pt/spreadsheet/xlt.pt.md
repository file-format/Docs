## O que é um arquivo XLT?

Arquivos com extensão .xlt são arquivos de modelo criados com o Microsoft Excel, que é um aplicativo de planilha que vem como parte do pacote Microsoft Office. O Microsoft Office 97-2003 suportava a criação de novos arquivos XLT, bem como a abertura deles. A versão mais recente do Excel ainda é capaz de abrir esses arquivos de modelo de formato antigo. Esse arquivo de modelo é usado para criar rapidamente novos arquivos do Excel com dados e configurações padrão, como formatação de página, tamanho da fonte, margens, gráficos, etc., que podem ser salvos como novos arquivos [.xls](/pt/planilha/xls/).

## Formato de arquivo XLT

O formato de arquivo XLT é baseado no formato de arquivo binário do Excel que inicialmente era compatível com a versão Excel 97-2003. Ele foi substituído pelo formato de arquivo [XLTX](/pt/spreadsheet/xltx/) com a introdução do Excel 2007, que é baseado no formato Office Open XML. Além do Microsoft Excel, existem outros aplicativos que suportam trabalhar com este formato de arquivo, como Lotus Translation Table, PlanMaker, LibreOffice Calc, etc. Vários aplicativos e [APIs](https://products.aspose.com/cells) estão disponíveis para converter XLT para outros formatos de arquivo, como:

* XLT para PDF
* XLT para DOC
* XLT para texto
* XLT para Lótus
* XLT para ODT
* XLT para ODS
* XLT para HTML
* XLT para CSV

XLT é baseado no binário [especificações de formato de arquivo XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) que é documentado pela Microsoft para referência do desenvolvedor.
