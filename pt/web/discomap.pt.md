## O que é um arquivo DISCOMAP?

Um arquivo DISCOMAP é um arquivo de saída criado pela Microsoft Web Services Discovery Tool, **Disco.exe**. Ele contém os resultados que são ASP.NET Web Services descobertos em uma URL específica. Exemplos comuns de tais serviços descobertos incluem locais de arquivos .DISCO ou .WSDL. Quando a ferramenta Microsoft Web Discovery é executada, ela procura informações de descoberta com base no conteúdo XML encontrado no arquivo DISCO. Os resultados são armazenados no arquivo DISCOMAP que pode ser aberto com o Microsoft Visual Studio IDE.
