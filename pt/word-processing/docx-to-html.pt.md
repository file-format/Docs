## Sobre DOCX para HTML

DOCX para HTML é uma das operações amplamente utilizadas em documentos do Word para publicar páginas como páginas HTML. Essas páginas da web podem ser facilmente usadas para serem hospedadas online em um servidor da web para serem acessíveis via World Wide Web (WWW). Você pode converter [DOCX](/pt/word-processing/docx/) em [HTML](/pt/web/html/) no software Microsoft Word usando a opção "Salvar como". Mas se você não tiver o Word instalado em seu PC, existem ferramentas online disponíveis que podem converter seus arquivos DOCX em HTML GRATUITAMENTE.
