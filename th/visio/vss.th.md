## ไฟล์ VSS คืออะไร??

VSS เป็นไฟล์สเตนซิลที่สร้างด้วย Microsoft Visio 2007 และรุ่นก่อนหน้า รูปแบบไฟล์ที่ค่อนข้างใหม่คือ [VSSX](/th/image/vssx/) ที่นำมาใช้กับ Microsoft Visio 2013 ไฟล์ลายฉลุมีวัตถุรูปวาดที่สามารถรวมอยู่ในรูปวาด .VSD Visio Microsoft Visio เองเป็นที่รู้จักในด้านการสร้างองค์ประกอบการวาด เช่น ชุดของรูปร่าง ตัวเชื่อมต่อ ผังงาน เค้าโครงเครือข่าย ไดอะแกรม UML ไดอะแกรมซอฟต์แวร์ แบบจำลองฐานข้อมูล การแมปวัตถุ และข้อมูลอื่นๆ ที่คล้ายคลึงกัน นอกจากนี้ยังมีคุณสมบัติการแปลงเอกสาร Visio ที่หลากหลายเป็นรูปแบบไฟล์อื่นๆ เช่น PNG, BMP, PDF และอื่นๆ Visio พร้อมใช้งานสำหรับทั้ง Windows และ Mac OS

## รูปแบบไฟล์ VSS

ซึ่งแตกต่างจาก VSSX ซึ่งใช้รูปแบบไฟล์ Office Open XML ไฟล์ VSS จะยึดตามรูปแบบไฟล์ไบนารี

## ประวัติย่อ ##

เริ่มแรก Visio ผลิตเป็นผลิตภัณฑ์แบบสแตนด์อโลนโดย Shapeware Corporation ในปี 1992 โดยเป็นเวอร์ชัน 1.0 Shapeware Corporation เปลี่ยนชื่อเป็น Visio Corporation ในปี 1995 เพื่อใช้ประโยชน์จากการยอมรับของตลาด Microsoft รีแบรนด์ Vision ในปี 2000 เป็นแอปพลิเคชัน Microsoft Office Visio for Enterprise Architects เปิดตัวพร้อมกับ Visual Studio .NET ในปี 2546 และ 2548 Visio 2007 เปิดตัวเมื่อวันที่ 30 พฤศจิกายน 2549 Microsoft Visio เปิดตัว Ribbon ในส่วนติดต่อผู้ใช้ในปี 2553
