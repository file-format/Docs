## ไฟล์ VSX คืออะไร??

ไฟล์ที่มีนามสกุล .vsx หมายถึงสเตนซิลที่ประกอบด้วยภาพวาดและรูปร่างที่ใช้สำหรับสร้างไดอะแกรมใน Microsoft Visio ไฟล์ VSX ถูกบันทึกในรูปแบบไฟล์ XML และรองรับจนถึง Visio 2013 ไฟล์เหล่านี้แตกต่างจากรูปแบบไฟล์ [VSDX](/th/image/vsdx/) หลักที่นำมาใช้กับ Microsoft Visio 2013 ไฟล์ VSX สามารถเปิดได้ในโปรแกรมแก้ไขข้อความใดๆ เพื่อดูเนื้อหา ไฟล์ VSX สามารถแปลงเป็นรูปแบบไฟล์ต่างๆ เช่น PDF และ HTML

## รูปแบบไฟล์ VSX ##

ตัวอย่างไฟล์ VSX ที่เปิดในโปรแกรมแก้ไข XML มีดังต่อไปนี้ ไฟล์ VSX ทั่วไปมีโหนดต่อไปนี้ตามที่สังเกตจากการแทนค่า XML

### คุณสมบัติเอกสาร ###

องค์ประกอบบางอย่างที่มีอยู่ในส่วนคุณสมบัติของเอกสารมีดังนี้:

```
<Creator>Creation Information</Creator>
<BuildNumberCreated>939530118</BuildNumberCreated>
<BuildNumberEdited>939530118</BuildNumberEdited>
<PreviewPicture Size#"20172">...</PreviewPicture>
<CustomProps>
<CustomProp Name#"_VPID_EXTENDED_VDX" PropType#"Number">1</CustomProp>
</CustomProps>
<TimeCreated>2012-07-06T23:00:56</TimeCreated>
<TimeSaved>2017-06-11T16:06:08</TimeSaved>
<TimeEdited>2012-07-06T23:06:33</TimeEdited>
<TimePrinted>2012-07-06T23:00:56</TimePrinted>
```

### สไตล์ชีต ###

โหนด StyleSheets มีข้อมูลเกี่ยวกับสไตล์ชีตที่ใช้ในเอกสาร แต่ละสไตล์ชีตสามารถมีข้อมูลเกี่ยวกับคุณสมบัติของสไตล์ เส้น รูปแบบการเติม บล็อคข้อความ และองค์ประกอบอื่นๆ

### แผ่นเอกสาร ###

โหนด DocumentSheet ประกอบด้วยข้อมูลเกี่ยวกับคุณสมบัติของเอกสารและข้อมูลชื่อผู้ใช้ และมีลักษณะดังต่อไปนี้:

```
<DocProps>
<OutputFormat>0</OutputFormat>
<LockPreview>0</LockPreview>
<AddMarkup>0</AddMarkup>
<ViewMarkup>0</ViewMarkup>
<PreviewQuality>0</PreviewQuality>
<PreviewScope>0</PreviewScope>
<DocLangID>1033</DocLangID>
</DocProps>
<User NameU#"msvNoAutoConnect" ID#"1">
<Value>1</Value>
<Prompt F#"No Formula"/>
</User>
</DocumentSheet>
```

### หน้า ###

มีข้อมูลเกี่ยวกับหน้าเอกสาร Visio เช่น คุณสมบัติหน้า เค้าโครง เส้นตารางไม้บรรทัด รูปร่าง รูปภาพ และข้อมูลอื่นๆ ที่คล้ายกัน
