## ไฟล์ VSSX คืออะไร??

ไฟล์ที่มีนามสกุล .vssx คือการวาดสเตนซิลที่สร้างด้วย [Microsoft Visio](https://products.office.com/en/visio/flowchart-software) 2013 ขึ้นไป รูปแบบไฟล์ VSSX สามารถเปิดได้ด้วย Visio 2013 ขึ้นไป ไฟล์ Visio เป็นที่รู้จักสำหรับการแสดงองค์ประกอบการวาดที่หลากหลาย เช่น ชุดของรูปร่าง ตัวเชื่อมต่อ ผังงาน เค้าโครงเครือข่าย ไดอะแกรม UML ไดอะแกรมซอฟต์แวร์ แบบจำลองฐานข้อมูล การแมปวัตถุ และข้อมูลอื่นๆ ที่คล้ายคลึงกัน Microsoft Visio ยังมีความสามารถในการแปลงไฟล์ Visio เป็นรูปแบบไฟล์ต่างๆ เช่น [PNG](/th/image/png/), [BMP](/th/image/bmp/), [PDF](/th/pdf/) และอื่นๆ ใช้ได้ทั้ง Windows และ Mac OS

## รูปแบบไฟล์ VSSX

รูปแบบไฟล์ VSSX อิงตามรูปแบบ OpenOffice ที่ Microsoft นำมาใช้ตั้งแต่ปี 2550 โดยยึดตามไฟล์เก็บถาวร [ZIP](/th/compression/zip/) ที่แสดงถึงรูปแบบไฟล์โดยรวมตามข้อกำหนดรูปแบบไฟล์ XML รูปแบบไฟล์ไบนารีที่เทียบเท่ากับ VSSX คือ VSS ที่ได้รับการสนับสนุนจนถึง Visio 2007 คุณสามารถดูเนื้อหาของรูปแบบไฟล์ VSSX ได้โดยแทนที่นามสกุลด้วย .ZIP และเปิดในรูปแบบไฟล์เก็บถาวรใดๆ เช่น WinZIP

ไฟล์ VSDX ขึ้นอยู่กับ Open Packaging Conventions และ XML และนักพัฒนาสามารถได้รับประโยชน์จากรูปแบบนี้โดยเรียนรู้วิธีการทำงานกับไฟล์เหล่านี้โดยทางโปรแกรม รูปแบบสืบทอดโครงสร้าง XML เดียวกันหลายส่วนจากรูปแบบไฟล์ Visio XML Drawing (.vdx) ความสามารถในการทำงานร่วมกันกับไฟล์ Visio นั้นเพิ่มขึ้นอย่างมาก เนื่องจากซอฟต์แวร์ของบริษัทอื่นสามารถจัดการไฟล์ Visio ในระดับรูปแบบไฟล์ได้

ไฟล์ประเภทอื่นๆ บางประเภทที่ประกอบด้วยรูปแบบไฟล์ Visio 2013 ได้แก่:

* .vsdm (การวาดภาพที่เปิดใช้งานมาโคร Visio)
* .vssx (ลายฉลุ Visio)
* .vssm (สเตนซิลที่เปิดใช้งานมาโคร Visio)
* .vstx (เทมเพลต Visio)
* .vstm (เทมเพลตที่เปิดใช้งานมาโคร Visio)

ไฟล์ Visio แต่ละไฟล์เรียกว่าแพ็คเกจที่เก็บไฟล์หรือส่วนอื่นๆ ส่วนของแพ็คเกจสามารถเป็นไฟล์ XML รูปภาพ หรือแม้แต่โซลูชัน VBA ส่วนต่างๆ ภายในแพ็คเกจสามารถแบ่งออกเป็นส่วน "เอกสาร" และ "ความสัมพันธ์"
