## ไฟล์ FODS คืออะไร??

ไฟล์ที่มีนามสกุล .fods คือรูปแบบเอกสารสเปรดชีต OpenDocument ประเภทหนึ่งที่เก็บข้อมูลในแถวและคอลัมน์ รูปแบบถูกระบุเป็นส่วนหนึ่งของข้อกำหนด ODF 1.2 ที่เผยแพร่และดูแลโดย OASIS ไม่สามารถเปิดไฟล์ FODS ด้วย Excel ซึ่งเป็นแอปพลิเคชันซอฟต์แวร์สเปรดชีตอื่นโดย Microsoft ไฟล์ FODS สามารถบันทึกเป็น [ODS](/th/spreadsheet/ods/) โดยใช้ LibreOffice และสามารถแปลงเป็นรูปแบบอื่น เช่น [XLS](/th/spreadsheet/xls) และ [XLSX](/th/spreadsheet/xlsx)

## รูปแบบไฟล์ FODS

OpenDocument กำหนดมาตรฐาน [ZIP](/th/compression/zip/) รวมถึง [XML](/th/web/xml/) สำหรับรูปแบบไฟล์ที่รองรับ รูปแบบไฟล์ FODS อ้างอิงจากรูปแบบเอกสาร XML เดียว หรือที่เรียกว่า Flat XML หรือไฟล์ XML ที่ไม่มีการบีบอัด การใช้รูปแบบ zipped เป็นเรื่องปกติมากกว่ารูปแบบ OpenDocument ที่ใช้ XML เช่นใน [ODS](/th/spreadsheet/ods/) ข้อกำหนดทางเทคนิคของ OpenDocument ไม่ได้แสดงรายการเอกสาร XML เดี่ยวใดๆ และไม่มีรายละเอียดเพิ่มเติมของเอกสารดังกล่าว
